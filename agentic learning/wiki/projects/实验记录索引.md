# 实验记录索引

这里记录小实验。实验不需要复杂，只要能验证一个概念边界。

## 最小实验原则

- 一个实验只验证一个问题。
- 先写预期，再看结果。
- 失败比成功更值得记录。

## 边界

概念触发式复习不是实验记录。它不按日期打卡，也不验证外部系统行为；它检查“我是否能解释清楚一个概念”。这类记录放在 [[reviews/复习记录索引]]。

## 可以做的第一个实验

问题：LLM 和 Agent 的区别在哪里？

实验：

1. 让 LLM 解释一个概念。

3. 对比两者：一个是生成回答，一个是读取环境并执行修改。
