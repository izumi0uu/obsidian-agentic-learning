# 复习记录索引

这里放概念触发式复习记录：先写出我自己的解释，再让 Codex 生成追问，用费曼方式回答，最后把暴露出来的边界、误解或新问题写回概念卡、[[02 问题池]] 或 [[05 Query 写回队列]]。

## 边界

- 复习记录不是 raw evidence，不承担“来源证明”。
- 复习记录不是概念卡，不要求一次写成稳定理解。
- 复习记录也不是固定 daily 打卡；它由正在学习的概念触发。
- 复习追问最多两轮：第一轮暴露误解，第二轮只处理关键卡点；第二轮后必须收束到写回候选、问题池或概念卡，不再继续加问。
- 追问卡一旦回答完，每张卡都必须有写回归宿：成熟边界写回概念卡 / 对比页；不成熟但重要的缺口写入 [[02 问题池]] 或 [[05 Query 写回队列]]；不能只停在反馈区。
