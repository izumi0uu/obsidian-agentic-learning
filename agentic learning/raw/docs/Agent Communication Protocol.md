# Agent Communication Protocol

## 为什么收

ACP 是 Agent 互操作的另一条协议线，来自 BeeAI/IBM 生态，强调 RESTful API、多模态消息、同步/异步、streaming、发现和长任务。

## 一句话

ACP 是用于连接 Agent、应用和人类的开放通信协议。

## 先读什么

- Welcome：协议目标和支持能力。
- Message Structure：理解 Agent 之间传递的信息结构。
- Production-grade：看安全、扩展和 observability 边界。
