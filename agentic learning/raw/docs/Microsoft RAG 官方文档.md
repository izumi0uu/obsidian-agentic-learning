# Microsoft RAG 官方文档

## 为什么收

Microsoft 的 RAG 文档适合学习企业级 RAG 的工程视角：数据源、索引、检索、生成、评估和安全边界。

## 先读什么

- RAG 的整体流程
- Azure AI Search 的角色
- 数据准备和索引
- 检索质量和生成质量

## 一句话

企业级 RAG 不只是“向量检索 + LLM”，还包括数据治理、索引策略、检索质量、权限和评估。

## 可以拆成概念卡

- [[RAG]]
- index
- hybrid search
- vector search
- retrieval quality
- access control
