## 题目正文

### 4. 问：RPC 的超时、重试、熔断怎么配？

答：先按下游 P99 延迟设超时，再设置有限次重试并加抖动退避，避免放大故障；对非幂等写请求默认不自动重试。熔断要按错误率和慢调用触发，并配合半开探测恢复，防止长期黑洞。  
追问：为什么“重试次数越多越危险”，在高峰时如何止损？
