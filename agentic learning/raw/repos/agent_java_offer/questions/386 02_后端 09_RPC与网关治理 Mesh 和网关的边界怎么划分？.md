## 题目正文

### 4. 问：Mesh 和网关的边界怎么划分？

答：网关主要处理南北向流量（外部到内部），Mesh 主要处理东西向流量（服务间通信）。两者都可做流量治理，但职责不同：网关更偏入口安全与聚合，Mesh 更偏服务间治理一致性。  
追问：同一条流量在网关和Mesh都限流会有什么问题？
