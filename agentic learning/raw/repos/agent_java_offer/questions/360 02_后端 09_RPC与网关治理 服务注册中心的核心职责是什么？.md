## 题目正文

### 3. 问：服务注册中心的核心职责是什么？

答：核心是“服务实例元数据维护 + 健康检查 + 发现路由”。提供者注册并周期上报心跳，消费者订阅实例变更并更新本地路由表。注册中心本身要保障高可用、弱依赖和故障时可降级读本地缓存。  
追问：注册中心短时不可用时，消费者如何避免全链路雪崩？
