## 题目正文

### 3. 问：分片键怎么选才能避免热点？

答：分片键要兼顾写入均匀和查询命中。优先选择高基数、稳定且查询常带条件的字段；避免时间戳这类天然热点键。可用“业务键 + 随机扰动”降低热点，并结合热分片迁移机制。  
追问：如果业务天然按商户聚集，热点商户怎么做专项治理？
