## 题目正文

### 1. Redis 为什么快？

核心原因是基于内存、单线程事件循环减少锁竞争、并使用高效数据结构（哈希表、跳表、压缩结构）。再叠加 I/O 多路复用和合理 pipeline，单机可实现很高吞吐和低延迟。
