## 题目正文

### 5. 问：零拷贝和 mmap 有什么价值？

答：核心是减少用户态与内核态之间的数据拷贝和上下文切换。sendfile/mmap 能显著提升大文件传输和高吞吐场景性能。Kafka、Nginx 等都依赖这些能力，但要注意页缓存行为和内存占用。  
追问：零拷贝能否完全避免 CPU 参与，为什么？
