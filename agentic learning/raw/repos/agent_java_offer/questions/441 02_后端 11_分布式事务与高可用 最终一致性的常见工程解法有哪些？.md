## 题目正文

### 2. 问：[[Durable Execution|最终一致性]]的常见工程解法有哪些？

答：最常见是 Outbox + MQ + 幂等消费 + 定时对账补偿。核心思想是把跨服务事务拆成可重试、可追踪、可回放的本地事务链，避免分布式全局锁。要保障消息可靠投递和重复消费安全。  
追问：Outbox 表积压后会带来什么风险，如何治理？
