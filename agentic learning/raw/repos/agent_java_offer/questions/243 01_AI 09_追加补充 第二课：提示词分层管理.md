## 题目正文

### **第二课：提示词分层管理**

Claude Code 的做法完全不同：它把提示词拆成静态层和动态层，静态部分走缓存节省费用，动态部分每次注入实时环境信息；每个工具还配一份专门给 AI 看的"使用手册"，明确告诉模型这个工具能干什么、绝对不能干什么。
