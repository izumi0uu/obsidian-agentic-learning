## 题目正文

### 11. RDB 与 AOF 取舍？

RDB 是全量快照存储，文件紧凑、恢复快，但可能丢失最近一段数据；AOF 记录写命令存储，数据更完整但文件更大、恢复更慢。生产常用混合策略：RDB 做基线，AOF 做增量保障。
