## 4. 补充原问：意图识别准确率如何定义与计算？

### 1. 子问题：意图识别准确率大概能到多少？怎么算出来的？

**口述答案（约300字）**：
准确率我不会给拍脑袋数字，必须先定义口径。通常我会按离线和在线两套指标：离线看标注测试集上的Accuracy、F1、混淆矩阵；在线看业务口径的任务完成率、误路由率和人工接管率。比如“准确率95%”要明确是Top1还是TopK、是否包含低置信度拒答样本。计算流程是：先构建分层数据集（高频、长尾、对抗样本），再做固定版本评测；上线后通过埋点还原“预测意图-真实落地结果”，做按周回归。对面试官我会强调：单看准确率不够，要看错误代价。如果一个低频误判会触发高风险动作，就要提高阈值并强制二次确认，宁可多澄清一次也不能错执行业务。
