## 题目正文

### **第一课：[[Agent|agent]]的工程特别重要，不不光光全是模型的能力，工程设计也很重要。**

Claude Code 的源码——51万行代码里，真正调用 [[LLM]] API 的部分不到 5%。剩下那 95% 是什么？是安全检查、权限管理、上下文压缩、错误恢复、[[Multi-agent Orchestration|多 Agent]] 协调。

换句话说，模型只是发动机，你造的是整辆车。发动机再好，没有底盘、没有刹车、没有方向盘，车照样跑不起来。
