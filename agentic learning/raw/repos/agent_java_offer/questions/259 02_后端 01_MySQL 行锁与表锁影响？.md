## 题目正文

### 10. 行锁与表锁影响？

行锁粒度细、并发高，适合 OLTP；表锁粒度粗，实现简单但会显著降低并发吞吐。实际排障里要重点看是否因为索引失效导致“行锁退化为大量行冲突”。
