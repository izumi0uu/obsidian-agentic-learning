## 题目正文

### 7. 覆盖索引和回表区别？

InnoDB 结构记住：
1.主键索引（聚簇索引）叶子节点存“整行数据”
2.二级索引叶子节点存“索引列 + 主键值”，不存整行
覆盖索引是查询字段都在二级索引里，直接从索引页返回结果，不需要再查主键。回表是先走二级索引拿主键，再去聚簇索引取完整行，I/O 次数和延迟都更高。
