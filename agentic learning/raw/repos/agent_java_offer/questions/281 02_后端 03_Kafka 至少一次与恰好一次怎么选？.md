## 题目正文

### 1. 至少一次与恰好一次怎么选？

恰好一次语义实现复杂、成本高，跨系统链路很难端到端保证。多数业务采用“至少一次投递 + 消费端幂等”，在可控复杂度下获得足够的可靠性。
