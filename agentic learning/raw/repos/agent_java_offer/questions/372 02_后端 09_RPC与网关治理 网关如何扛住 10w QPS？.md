## 题目正文

### 3. 问：网关如何扛住 10w QPS？

答：核心是无状态横向扩容 + 轻量过滤链 + 连接池优化。数据面用 Nginx/Netty 低开销处理，热点校验本地缓存化；慢逻辑异步化。再配合 CPU 绑核、零拷贝、限流和熔断，保障高峰时延可控。  
追问：在网关里做 JWT 鉴权时，如何降低验签 CPU 消耗？
