## 题目正文

### 4. redis的哨兵机制

解决主从模式master宕机后服务不可用的问题, 是一个独立的进程,监控多个master-slave集群,通过多哨兵投票确认故障并自动选主切换，保证服务在主节点故障后可恢复.
高可用用 Sentinel；要横向扩容/分片用 Redis Cluster;
