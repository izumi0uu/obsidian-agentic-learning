## 题目正文

### 4. 为什么自调用会让 `@Transactional` 失效？

Spring 事务基于代理拦截，方法自调用走的是对象内部调用，不经过代理，所以事务注解不生效。常见解法是拆到独立 Bean、通过代理对象调用。
