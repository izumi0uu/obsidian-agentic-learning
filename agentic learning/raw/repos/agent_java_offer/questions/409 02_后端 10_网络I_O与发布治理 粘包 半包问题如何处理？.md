## 题目正文

### 4. 问：粘包/半包问题如何处理？

答：TCP 是字节流，不保留消息边界，可能出现粘包半包。解决方式是自定义协议边界：固定长度、分隔符、长度字段。Netty 常配合 LengthFieldBasedFrameDecoder 等解码器确保消息按帧还原。  
追问：长度字段被恶意构造时，如何防止内存攻击？
