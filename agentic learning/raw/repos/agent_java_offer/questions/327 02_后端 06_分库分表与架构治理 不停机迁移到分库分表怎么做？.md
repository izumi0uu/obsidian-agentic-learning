## 题目正文

### 6. 问：不停机迁移到分库分表怎么做？

答：通常走“双写 + 回填 + 校验 + 切流 + 回滚”五步。先建立新链路并双写，历史数据离线回填，做全量与增量校验后灰度切读切写，保留回滚开关。关键是[[Observability|可观测]]和幂等补偿链路完备。
追问：双写阶段如何避免主从数据长期漂移？

高频连问路径：

1. 拆分时机 -> 拆分方式 -> 成本评估
2. 分片键 -> 热点治理 -> 扩容迁移
3. 跨分片查询 -> 聚合策略 -> 数据一致性
