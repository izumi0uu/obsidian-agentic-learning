## 题目正文

### 2. 问：垂直拆分和水平拆分怎么选？

答：垂直拆分按业务域解耦，解决的是职责和团队边界问题；水平拆分按数据量扩展，解决的是容量和吞吐问题。常见路径是先垂直再水平，避免把单体问题直接放大成分布式问题。
