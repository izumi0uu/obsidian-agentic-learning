## 题目正文

### 1. 问：网关的职责边界是什么？

答：网关负责统一入口治理：鉴权、路由、限流、灰度、[[Audit Log|审计]]和协议转换；不应承载重业务逻辑。原则是“控制面集中、业务面下沉”，避免网关演化成超级单体，导致发布风险和性能瓶颈同时上升。
追问：哪些逻辑必须放网关，哪些必须下沉到业务服务？
