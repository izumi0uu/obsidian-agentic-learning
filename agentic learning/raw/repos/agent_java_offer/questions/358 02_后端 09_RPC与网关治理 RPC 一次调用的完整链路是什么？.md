## 题目正文

### 2. 问：RPC 一次调用的完整链路是什么？

答：调用方先通过代理拦截本地接口，序列化请求并做服务发现、负载均衡，再经网络传输到服务端；服务端反序列化后执行业务逻辑并返回结果，客户端再做反序列化、超时控制和[[Durable Execution|重试]]决策。
追问：链路里最常见的性能瓶颈在哪三处，如何快速定位？
