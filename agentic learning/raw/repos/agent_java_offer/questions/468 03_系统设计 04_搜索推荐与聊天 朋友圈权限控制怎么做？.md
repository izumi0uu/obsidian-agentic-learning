## 题目正文

### 朋友圈权限控制怎么做？

权限本质上是“关系 + 可见范围 + 用户配置”的联合判断。

一种可讲的工程化思路是：
- 用户和好友之间的关系、朋友圈权限设置、总可见范围这些信息，都需要在服务端持久化；
- 同时客户端会缓存一部分权限数据，用于本地快速判断和展示优化；
- 每次刷新朋友圈时，再结合当前用户与发帖人的关系、帖子本身的可见范围，做一次最终判定。

一句话说：
**发布时控制可分发范围，读取时再做一次[[Tool Permissioning|权限校验]]，形成双层控制。**
