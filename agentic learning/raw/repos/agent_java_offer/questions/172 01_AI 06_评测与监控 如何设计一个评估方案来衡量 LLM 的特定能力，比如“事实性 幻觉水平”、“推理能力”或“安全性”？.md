## 题目正文

### 5. 子问题：如何设计一个评估方案来衡量 LLM 的特定能力，比如“事实性/幻觉水平”、“推理能力”或“安全性”？

答：

**1.衡量“事实性/幻觉水平”：**

- **就是基于一个知识库进行问答。然后看这个模型所回答的是否能从这个知识库中找到答案进行匹配，不是自己瞎编的。另外就是设计一些能够以对抗性的问题，能够诱导模型产生幻觉的问题。比如说询问不存在的人或者事件的信息，然后看一下模型的回答。**

**2. 衡量“推理能力”：**

- 那推理能力就是让模型做一些数，数学题或者逻辑推理的题，或者需需要自行设计一些推理的任务。然后呢，一评估的时候，一个是看结果的评估，就看最后这个回答的是否是准确。另外一个是过程的评估，就是看模型的思维链。要主要看它的推理步骤是否合乎逻辑，是否正确和推理过程。

**3. 衡量“安全性”：**

- 安全性就是模模型是否会拒绝一些回答有害、不道德或者危险的一些用户请求。所以就要找一些对抗性的提示词数据集，然后包含了一些精心设计的引导模型绕过安全护栏的危险问题。另外一个思路就是专门的红蓝攻防，由人类专家主动的构造性的做一些攻击性的提示。评判方法就是看这个模型的一个拒绝率。模型看拒绝回，要拒绝回答这个有害问题的比例。以及误伤率，就是模型会错误的拒绝回答一个正常安全的一个问题。还有就是人工评估介入审核。
