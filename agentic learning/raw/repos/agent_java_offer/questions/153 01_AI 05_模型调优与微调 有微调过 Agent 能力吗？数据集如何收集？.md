## 题目正文

### 2. 子问题：有微调过 Agent 能力吗？数据集如何收集？

答：
Agent 微调的重点不是“最终答案”，而是“决策轨迹”, **教会模型如何更好地“思考”和“[[Tool Use|使用工具]]”**，本质上是一种**行为克隆（Behavioral Cloning）.**

常见做法是让强教师模型生成高质量 Thought/Action/[[Observation]] **决策**轨迹，再过滤失败样本，结合人工修正构造训练集。

数据来源一般包括合成任务、真实线上失败案例和专家示范。实践里先做行为克隆打基础，再用偏好数据做策略优化，提升稳定性。
