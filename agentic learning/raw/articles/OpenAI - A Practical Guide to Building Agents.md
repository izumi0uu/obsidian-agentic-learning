## 为什么收

这是 OpenAI 面向构建 Agent 的实践指南，适合作为从概念过渡到工程判断的资料。

## 先读什么

- Agent 的定义

## 一句话

构建 Agent 的重点不是让模型自由发挥，而是给它明确任务、工具、边界和评估。
