# 08 面试题概念卡待补充

## 使用规则

- 本页只记录面试题中出现、但尚未确认稳定 canonical name 或边界的概念候选。
- 候选进入本页不等于应该新建概念卡；先补证据、术语边界和中英文对齐，再决定新建、合并、拒绝或继续挂起。
- 不在本页写完整概念解释；稳定理解应写回 `wiki/concepts/`、`wiki/topics/` 或 [[05 Query 写回队列]]。
- 面试题 raw/source 正文不因本页候选而被改写；只允许在明确匹配后补 `related` 或 `## 相关知识 wiki`。
